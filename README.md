# Economic Models

A collection of applied economic modeling projects covering macroeconomics, microeconomics, and finance, with reproducible Python workflows, public data, and clear analytical reports.

## Repository Structure

- `macro/`: macroeconomic models and empirical analysis.
- `micro/`: microeconomic models and applied micro projects.
- `finance/`: financial models, asset pricing, risk analysis, and market data projects.

## Projects

### Macroeconomics

- `us_phillips_curve`: analysis of the US Phillips Curve using FRED data, including inflation, unemployment, stationarity tests, linear and quadratic specifications, and a stakeholder-oriented report.

## Reproducibility

Each project includes its own files and instructions when needed. Most projects use Python, public datasets, and reproducible workflows.
